Marco Nie - ssh 2019-03-21T20:28:21+08:00 Typecho https://blog.niekun.net/feed/atom/tag/ssh/ <![CDATA[修改SSH端口]]> https://blog.niekun.net/archives/135.html 2019-03-21T20:28:21+08:00 2019-03-21T20:28:21+08:00 admin https://niekun.net ssh 默认端口为 22， 有时候为了安全，需要修改默认端口到任意其他端口。

使用 root 用户登录服务器。

ssh root@yourip

打开配置文件：

vim /etc/ssh/sshd_config

找到并修改以下行：

Port 22

修改 22 为任意可用端口，保存并退出。

重启服务：

service ssh restart

重新登录 ssh：

ssh root@yourip -p Port

以上就是修改 ssh 端口的过程。
