Linux 查看端口占用参考：https://niekun.net/index.php/archives/430.html

使用命令如下：

# 显示所有端口情况
netstat -ano

# 显示某个端口的占用情况 返回的数据中最后的数字为 PID 号
netstat -aon|findstr "7200"

# 根据 PID 号结束对应进程
taskkill /f /pid  7200

查看当前所有后台进程：

ps -aux | grep "test.sh"

关闭进程，通过ps命令查看进程号PID，然后执行：

kill %PID

lsof

查看当前端口是否被某个进程占用：

lsof -i:8000
