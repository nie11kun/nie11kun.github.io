使用 root 用户登录服务器。

ssh root@yourip

打开配置文件：

vim /etc/ssh/sshd_config

找到并修改以下行：

Port 22

修改 22 为任意可用端口，保存并退出。

重启服务：

service ssh restart

重新登录 ssh：

ssh root@yourip -p Port

以上就是修改 ssh 端口的过程。
