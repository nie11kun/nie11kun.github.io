Marco Nie - ftp https://blog.niekun.net/tag/ftp/ zh-CN Mon, 18 Feb 2019 15:14:00 +0800 Mon, 18 Feb 2019 15:14:00 +0800 使用Nginx搭建HTTP共享文件夹 https://blog.niekun.net/archives/26.html https://blog.niekun.net/archives/26.html Mon, 18 Feb 2019 15:14:00 +0800 admin 最近有需求做一个类似网盘的功能，方便共享文件给他人，实际上就是把服务器的一个目录设置为可见。发现Nginx支持这一功能。

官方相关文档：http://nginx.org/en/docs/http/ngx_http_autoindex_module.html

在service里的新建需要共享目录的location段修改如下：

location /downloads {
    root /home;
    autoindex on;
    autoindex_exact_size on;
    autoindex_localtime on;
}

后三句为打开当前location段的目录索引功能，显示文件大小，显示文件创建日期，建议只对需要共享的目录打开目录功能。
