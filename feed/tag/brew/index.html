Marco Nie - brew https://blog.niekun.net/tag/brew/ zh-CN Thu, 21 Feb 2019 20:51:00 +0800 Thu, 21 Feb 2019 20:51:00 +0800 macOS使用brew安装的模块提示命令不存在的解决方法 https://blog.niekun.net/archives/100.html https://blog.niekun.net/archives/100.html Thu, 21 Feb 2019 20:51:00 +0800 admin 此问题主要是没有把 /usr/local/sbin 加入path，使用下面指令可以将其加入path：

export PATH=$PATH:/usr/local/sbin

此指令只对当前终端窗口有效，关闭后失效。可将命令加入 ~/.bash_profile 文件使设置永久有效,此文件一般不存在，可新建：

vim ~/.bash_profile

加入代码：

export PATH=$PATH:/usr/local/sbin

关闭文件后，重新载入配置：

source ~/.bash_profile

重新运行命令测试是否已经可用。
