Marco Nie - blend https://blog.niekun.net/tag/blend/ Blend Mode 混合模式 in Photoshop https://blog.niekun.net/archives/563.html 2019-12-02T13:49:44+08:00 Photoshop 中的混合模式是使用很频繁的功能，用好了解各个混合的原理对于图像处理有很大的帮助。 ![2019-12-02T02:57:22.png][1] ## normal group ### normal 上面的图层简单的覆盖下面的图层部分： ![2019-12-02T02:59:11.png][2] ### dissolve dissolve 和 normal 类似，只是在调节不透明度的时候，透明部分是‘点’构成的： ![2019-12-02T03:02:08.png][3] ## darken group ### darken 当前图层的亮度低于背景图层的部分覆盖到背景，表现为背景图层暗部变暗： ![2019-12-02T03:04:38.png][4] ### multiply 相比较于 darken，当前图层的亮度信息作为一个整体信息作用于背景图层，当前图层亮度100% 为1 亮度0% 为0，对背景图层每个像素乘以当前图层的亮度百分比，使整体背景变暗： ![2019-12-02T03:12:03.png][5] ### color burn 背景图层亮度为 100% 和 0% 的地方不做影响，类似 multiply，且影响 color： ![2019-12-02T03:40:16.png][6] ![2019-12-02T03:35:49.png][7] color burn 是 8 个特殊模式之一，所以调整填充和不透明度效果不一样： ![2019-12-02T03:38:08.png][8] ### linear burn 和 color burn 类似，但会影响亮度为 100% 和 0% 的地方： ![2019-12-02T03:41:52.png][9] ### darker color 类似与 darken，但是过度更加生硬： ![2019-12-02T03:43:24.png][10] ## lighten group ### lighten 是 darken 的反向，当前图层的亮度高于背景图层的部分覆盖到背景，表现为背景图层亮部变量： ![2019-12-02T03:45:19.png][11] ### screen 是 multiply 的反向，前景图层亮度为越高背景图层对应像素亮度越提高： ![2019-12-02T03:50:36.png][12] ### color dodge 是 color burn 的反向，功能是给背景增加颜色，提高亮度，但不影响亮度为 100% 和 0% 的地方： ![2019-12-02T03:56:47.png][13] ### linear dodge 相比较 color dodge 会影响阴影部分和高光部分： ![2019-12-02T03:58:05.png][14] ### lighter color 相比较 lighten 更加生硬： ![2019-12-02T03:59:23.png][15] ## contrast group ### overlay 增加背景图层对比度，前景图层亮度大于 50% 的地方提亮，低于 50% 的地方变暗： ![2019-12-02T04:02:56.png][16] ### soft light 类似于 overlay 对比度小一些： ![2019-12-02T04:59:57.png][17] ### hard light 相比较于 overlay 会使背景产生 fade 的效果，使图像更加柔滑： 复制背景，高斯模糊后选择 hard light： ![2019-12-02T05:02:34.png][18] ### vivid light 增加对比，增加颜色，不影响 100% 和 0% 亮度部分： ![2019-12-02T05:06:50.png][19] ### linear light： 类似 vivid light，会影响 100% 和 0% 亮度部分： ![2019-12-02T05:07:54.png][20] 可以用来跟图片增加颜色和对比度： ![2019-12-02T05:09:01.png][21] ### pin light 效果是 darken 和 lighten 的结合： ![2019-12-02T05:12:42.png][22] ### hard mix 将图层色彩转换为 8 种颜色：**r g b c m y white black** ![2019-12-02T05:15:21.png][23] ![2019-12-02T05:15:03.png][24] hard mix 属于 special blend mode，调节 fill 可以得到特殊效果： ![2019-12-02T05:20:08.png][25] ## inversion group 将图层间的 rgb 值比较差值，相同颜色差值为 0，显示为黑色： ![2019-12-02T05:24:39.png][26] 可以用来比较两张图片间不同的地方，相同的地方会显示黑色，差距越大越亮： ![2019-12-02T05:33:39.png][27] ![2019-12-02T05:34:02.png][28] ## component group ### hue hue 代表 root 颜色，改变背景图层的 root 颜色，不改变亮度 饱和度等信息，对于灰度图像没有影响： ![2019-12-02T05:40:10.png][29] ### saturation saturation 代表颜色的饱和度，改变背景图层的颜色饱和度，不改变 hue 亮度等信息，不影响灰度图像： ![2019-12-02T05:42:20.png][30] ### color color 类似于 hue，但是会影响灰度图像： ![2019-12-02T05:44:27.png][31] ### luminosity luminosity 代表亮度，改变背景图层的亮度信息，不影响颜色和饱和度： 使用曲线增加对比后，会引起饱和度的变化，把混合模式改为 luminosity 可以修复这一问题： ![2019-12-02T05:46:11.png][32] 使用 black & write 图层，设置 luminosity 模式，可以调节图片细节： ![2019-12-02T05:48:11.png][33] ## special blend mode 特殊混合模式，调节 fill 和 不透明度有不同的效果： ![2019-12-02T03:36:19.png][34] [1]:
