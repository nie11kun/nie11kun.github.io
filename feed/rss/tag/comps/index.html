Marco Nie - comps https://blog.niekun.net/tag/comps/ 使用 Layer Comps  实现一个图片文件多个效果版本 in Photoshop https://blog.niekun.net/archives/874.html 2020-02-12T09:57:00+08:00 ![2020-02-12T01:58:10.png][1] 一张图片 psd 项目可以随时切换多种风格的效果，可以在 psd 源文件内将多个风格的图层都建立起来，在 layer comps 做标记，然后在其他项目中调用此 psd 源文件时候就可以选择性的切换各种效果。 想在项目中插入另一个 psd 文件，选择 file - place linked： ![2020-02-12T01:29:04.png][2] 插入后，打开右侧 properties 选项，如果没有，从 Windows 里找到打开： ![2020-02-12T01:32:27.png][3] 如果打开的 psd 文件里已经定义了 payer comps，则从这个选项就可以切换了： ![2020-02-12T01:33:34.png][4] **下面介绍如何定义 layer comps** 首先将图片需要的各种风格的图层都建立好： ![2020-02-12T01:44:15.png][5] 选择 window - layer comps： ![2020-02-12T01:45:30.png][6] 根据想要的第一种风格，关闭打开相关图层，点击 layer comps 下面的加号，将所有选项都打勾，起一个名字，就完成了第一个 layer comps 的定义，点击确定，我这里第一个风格是原始图片： ![2020-02-12T01:48:32.png][7] 同理，建立想要的不同风格 layer comps： ![2020-02-12T01:51:30.png][8] 在前面的项目中打开我们刚才建立的包含 layer comps 的 psd 文件，选择 file - place linked： ![2020-02-12T01:55:25.png][9] 打开 properties 查看可选的 layer comps： ![2020-02-12T01:56:09.png][10] 点击不同的风格，可以看到效果了。 以上就是使用 layer comps 实现 psd 文件多种风格的定义。 [1]: https://niekun.net/usr/uploads/2020/02/291400954.png [2]: https://niekun.net/usr/uploads/2020/02/2065094755.png [3]: https://niekun.net/usr/uploads/2020/02/650889507.png [4]: https://niekun.net/usr/uploads/2020/02/2926031993.png [5]: https://niekun.net/usr/uploads/2020/02/1252132334.png [6]: https://niekun.net/usr/uploads/2020/02/3557653880.png [7]: https://niekun.net/usr/uploads/2020/02/1009476175.png [8]: https://niekun.net/usr/uploads/2020/02/2882900059.png [9]: https://niekun.net/usr/uploads/2020/02/2240620771.png [10]: https://niekun.net/usr/uploads/2020/02/3874024248.png
