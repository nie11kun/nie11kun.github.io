使网站支持http2

  • 作者: admin
  • 时间: 2019-04-03
  • 分类: Linux,web
  • 评论

这里使用的是 Nginx 做代理。

使用 http2 需要 openssl 的版本至少是 1.0.2。使用命令openssl version查看当前版本，如果太低，需要先升级 openssl。

升级教程：https://niekun.net/index.php/archives/30.html

在 Nginx 配置文件的 server 段内修改如下：

server {
    listen               443 ssl http2;
    listen               [::]:443 ssl http2;
    ...
}

实际就是在原有配置基础上，在 listen 内加入 http2 即可。

修改后测试及重新加载配置文件：

service nginx configtest
service nginx reload

修改SSH端口

  • 作者: admin
  • 时间: 2019-03-21
  • 分类: Linux
  • 评论

ssh 默认端口为 22， 有时候为了安全，需要修改默认端口到任意其他端口。

使用 root 用户登录服务器。

ssh [email protected]

打开配置文件：

vim /etc/ssh/sshd_config

找到并修改以下行：

Port 22

修改 22 为任意可用端口，保存并退出。

重启服务：

service ssh restart

重新登录 ssh：

ssh [email protected] -p Port

以上就是修改 ssh 端口的过程。
