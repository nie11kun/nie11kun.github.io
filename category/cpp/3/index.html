  • 2020-11-20
  • c++

一个 function 是为了完成某个任务的许多命令的一个集合。使用 function 的优势有：

  • 代码复用
  • 独立测试某功能
  • 修改一个 function 内部代码不影响程序整体结构
  • 同一个 function 可使用不同传入参数调用

一个有效的 c++ 程序至少要有一个 function：main()。
