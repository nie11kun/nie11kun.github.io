Blend Mode 混合模式 in Photoshop

  • 作者: admin
  • 时间: 2019-12-02
  • 分类: photoshop

Photoshop 中的混合模式是使用很频繁的功能，用好了解各个混合的原理对于图像处理有很大的帮助。

normal group

normal

上面的图层简单的覆盖下面的图层部分：

dissolve

dissolve 和 normal 类似，只是在调节不透明度的时候，透明部分是‘点’构成的：

darken group

darken

当前图层的亮度低于背景图层的部分覆盖到背景，表现为背景图层暗部变暗：

multiply

相比较于 darken，当前图层的亮度信息作为一个整体信息作用于背景图层，当前图层亮度100% 为1 亮度0% 为0，对背景图层每个像素乘以当前图层的亮度百分比，使整体背景变暗：

color burn

背景图层亮度为 100% 和 0% 的地方不做影响，类似 multiply，且影响 color：

color burn 是 8 个特殊模式之一，所以调整填充和不透明度效果不一样：

linear burn

和 color burn 类似，但会影响亮度为 100% 和 0% 的地方：

darker color

类似与 darken，但是过度更加生硬：

lighten group

lighten

是 darken 的反向，当前图层的亮度高于背景图层的部分覆盖到背景，表现为背景图层亮部变量：

screen

是 multiply 的反向，前景图层亮度为越高背景图层对应像素亮度越提高：

color dodge

是 color burn 的反向，功能是给背景增加颜色，提高亮度，但不影响亮度为 100% 和 0% 的地方：

linear dodge

相比较 color dodge 会影响阴影部分和高光部分：

lighter color

相比较 lighten 更加生硬：

contrast group

overlay

增加背景图层对比度，前景图层亮度大于 50% 的地方提亮，低于 50% 的地方变暗：

soft light

类似于 overlay 对比度小一些：

hard light

相比较于 overlay 会使背景产生 fade 的效果，使图像更加柔滑：
复制背景，高斯模糊后选择 hard light：

vivid light

增加对比，增加颜色，不影响 100% 和 0% 亮度部分：

linear light：

类似 vivid light，会影响 100% 和 0% 亮度部分：

可以用来跟图片增加颜色和对比度：

pin light

效果是 darken 和 lighten 的结合：

hard mix

将图层色彩转换为 8 种颜色：r g b c m y white black

hard mix 属于 special blend mode，调节 fill 可以得到特殊效果：

inversion group

将图层间的 rgb 值比较差值，相同颜色差值为 0，显示为黑色：

可以用来比较两张图片间不同的地方，相同的地方会显示黑色，差距越大越亮：

component group

hue

hue 代表 root 颜色，改变背景图层的 root 颜色，不改变亮度 饱和度等信息，对于灰度图像没有影响：

saturation

saturation 代表颜色的饱和度，改变背景图层的颜色饱和度，不改变 hue 亮度等信息，不影响灰度图像：

color

color 类似于 hue，但是会影响灰度图像：

luminosity

luminosity 代表亮度，改变背景图层的亮度信息，不影响颜色和饱和度：
使用曲线增加对比后，会引起饱和度的变化，把混合模式改为 luminosity 可以修复这一问题：

使用 black & write 图层，设置 luminosity 模式，可以调节图片细节：
