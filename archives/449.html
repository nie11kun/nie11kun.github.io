windows 安装最新版 openssl

2019-11-07 web，Windows，software 0
这篇文章上次修改于 1063 天前，可能其部分内容已经发生变化，如有疑问可询问作者。

最近折腾 vps 的自动备份，备份文件是在服务器上使用 openssl 加密的，我的一台笔记本是 Windows 系统的，所以想要解密的话需要在 windows 上安装 openssl，但是官方网站没有提供对应版本，做了一些查找。

在此网站下载需要版本的安装包：https://slproweb.com/products/Win32OpenSSL.html

安装过程不再介绍，在其中一个选项，选择将 openssl 安装到 /bin。

将 安装目录中的 /bin/openssl.exe 添加到系统 PATH 中,参考：https://niekun.net/index.php/archives/413.html
