hex <-> str in python

2020-08-12 python 0
这篇文章上次修改于 535 天前，可能其部分内容已经发生变化，如有疑问可询问作者。

str to hex:

import codecs
hexlify = codecs.getencoder('hex')
hexlify(b'Blaah')[0]

out:

b'426c616168'

hex to str:

import codecs
decode_hex = codecs.getdecoder("hex_codec")
s = decode_hex(b'426c616168')[0]

out:

b'Blaah'
