Lightroom 初始推荐设置

  • 作者: admin
  • 时间: 2020-02-16
  • 分类: lightroom

Lightroom 是一个集成照片管理/编辑的强大工具，在使用前进行一些自定义的设置，能够有效的提高效率。

选项卡设置为 solo mode

在编辑菜单中有很多不同的 table 分类，为了集中注意力到当前打开的 table，设置到 solo mode 可以关闭除当前打开之外的所有窗口：

照片以文件夹形势归类

一次拍摄的照片应该放在一个单独的文件夹，然后进行命名，方便日后管理：

文件重命名

首先想要在 grid 预览中显示图片文件名，选择 view - view option，选中 top lable：

文件夹导入 Lightroom 后，除了文件夹本身进行命名归类，里面的照片也应给重命名，这样便于单独搜索或者查看，ctrl A 全选文件夹内图片，选择 library - rename photos：
