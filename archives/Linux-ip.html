Linux 获取本地网关所有 ip 段地址

  • 2024-03-28
  • Linux, other
  • 暂无评论

执行下面命令即可获取本地网关所有的 ip 段地址：

$ ip address | grep -w inet | awk '{print $2}'
127.0.0.1/8
192.168.122.2/24
172.18.0.1/16
172.17.0.1/16
172.19.0.1/16
