通过 GitHub API 获取数据

2021-04-08 web 0
这篇文章上次修改于 683 天前，可能其部分内容已经发生变化，如有疑问可询问作者。

最近发现通过 GitHub API 可以获取到很多有用的数据，它返回一个 json 格式数据，可以后期解析得到需要的内容。

主链接：https://api.github.com/

返回内容如下：
