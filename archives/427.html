Linux 软链接 link

  • 作者: admin
  • 时间: 2019-10-16
  • 分类: Linux

类似于 Windows 下的快捷方式，Linux 下可以创建一个文件/文件夹的 link，用于某个文件/文件夹的快速访问，也可以用在自己安装的 app 能够使用命令调用，或者设置同一 app 不同版本的调用。

ln -s source_filename symbolic_filename

-s：软连接
