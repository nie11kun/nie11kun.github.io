React е…Ҙй—Ёж•ҷзЁӢд№ӢдәҢ -- hello world

  • 2021-02-24
  • react
  • жҡӮж— иҜ„и®ә

дёӢйқўжҲ‘д»¬йҖҡиҝҮеҲӣе»ә Create React App д»Ӣз»Қ React зҡ„еҹәжң¬иҜӯжі•гҖӮ

йҖҡиҝҮдёҠдёҖиҠӮд»Ӣз»Қзҡ„ж–№жі•еҲӣе»әдёҖдёӘж–°зЁӢеәҸ my-app-1:

npx create-react-app my-app-1
cd my-app-1

е…ідәҺеҲӣе»әж–°йЎ№зӣ®еҸҠеҸҜиғҪеҮәзҺ°зҡ„й—®йўҳеҸӮиҖғдёҠдёҖзҜҮпјҡhttps://blog.niekun.net/archives/2175.html

Create React App е®ҳж–№ж•ҷзЁӢпјҡhttps://create-react-app.dev/docs/documentation-intro

йЎ№зӣ®з»“жһ„

зЁӢеәҸзӣ®еҪ•з»“жһ„еҰӮдёӢпјҡ

my-app-1

еҜ№дәҺдёҖдёӘйЎ№зӣ®иҖҢиЁҖпјҢдёӢйқўдёӨдёӘж–Үд»¶жҳҜеҝ…йЎ»зҡ„:

  • public/index.html жҳҜйЎөйқўзҡ„жЁЎжқҝж–Үд»¶;
  • src/index.js жҳҜ JavaScript е…ҘеҸЈж–Үд»¶.

е…¶д»–ж–Үд»¶еҸҜд»Ҙж №жҚ®йңҖиҰҒж·»еҠ пјҢеҲ йҷӨжҲ–йҮҚе‘ҪеҗҚгҖӮ

еҸҜд»ҘеңЁ src зӣ®еҪ•дёӢе»әз«Ӣеӯҗзӣ®еҪ•пјҢдёәдәҶеҠ еҝ« build зј–иҜ‘йЎ№зӣ®зҡ„йҖҹеәҰпјҢеҸӘжңүеңЁ src зӣ®еҪ•дёӢзҡ„ж–Үд»¶жүҚдјҡиў« webpack еӨ„зҗҶгҖӮжүҖд»ҘйңҖиҰҒе°ҶжүҖжңүзҡ„ js е’Ң css ж–Үд»¶ж”ҫеңЁ src зӣ®еҪ•дёӢпјҢеҗҰеҲҷ webpack жү“еҢ…еҷЁдёҚдјҡиҜҶеҲ«еҲ°е®ғд»¬гҖӮ

еҸӘжңүеңЁ public зӣ®еҪ•дёӢзҡ„ж–Үд»¶жүҚдјҡиў« public/index.html ж–Үд»¶иҜҶеҲ«еҲ°гҖӮ

жүҖд»ҘжүҖжңүзҡ„йЎ№зӣ®ж–Үд»¶йғҪйңҖиҰҒж”ҫеңЁ public е’Ң src зӣ®еҪ•дёӢпјҢдёҺйЎ№зӣ®ж— е…ізҡ„ж–Үд»¶еҸҜд»Ҙж”ҫеңЁдёҠзә§и·Ҝеҫ„еҶ…пјҢдҫӢеҰӮйЎ№зӣ®иҜҙжҳҺж–Үд»¶зӯүгҖӮ

еҸҜз”Ёи„ҡжң¬жҢҮд»Ө

еңЁйЎ№зӣ®зӣ®еҪ•дёӢпјҢеҸҜд»Ҙжү§иЎҢеҰӮдёӢе‘Ҫд»ӨгҖӮ

npm start

ејҖеҸ‘жЁЎејҸдёӢиҝҗиЎҢ appпјҡ

npm start

жү“ејҖ http://localhost:3000 еҸҜд»ҘжҹҘзңӢж•ҲжһңгҖӮеҜ№д»Јз ҒдҪңеҮәдҝ®ж”№еҗҺйЎөйқўдјҡиҮӘеҠЁйҮҚж–°еҠ иҪҪгҖӮ

npm test

дәӨдә’жЁЎејҸдёӢиҝҗиЎҢ test иҝҗиЎҢеҷЁгҖӮжӣҙеӨҡд»Ӣз»ҚеҸӮиҖғпјҡhttps://create-react-app.dev/docs/running-tests

npm run build

зј–иҜ‘йЎ№зӣ®зЁӢеәҸеҲ° build зӣ®еҪ•дёӢпјҢдјҡиҮӘеҠЁе°Ҷ React иҪ¬жҚўдёәдә§е“ҒжЁЎејҸпјҢе№¶дјҳеҢ–йЎ№зӣ®з»“жһ„д»ҘжҸҗй«ҳжҖ§иғҪгҖӮд»Јз ҒжҳҜз»ҸиҝҮ minified жңҖе°ҸеҢ–еӨ„зҗҶиҝҮзҡ„пјҢж–Үд»¶еҗҚйғҪеҢ…еҗ« hashes д»ҘеңЁе®ўжҲ·з«ҜиғҪеӨҹеҸҠж—¶иҜҶеҲ«жӣҙж–°гҖӮ

жӣҙеӨҡ build еҸӮиҖғпјҡhttps://create-react-app.dev/docs/production-build

жё…зҗҶйЎ№зӣ®

й»ҳи®ӨдҪҝз”Ёзҡ„жЁЎжқҝжҳҜ cra-templateпјҢеҢ…еҗ«дёҖдёӘеҹәжң¬зҡ„йЎөйқўе’Ң React componentпјҢжҲ‘д»¬е…Ҳе°ҶйЎ№зӣ®жё…зҗҶжҺүдёҚйңҖиҰҒзҡ„еҶ…е®№гҖӮ

жү“ејҖ public зӣ®еҪ•дёӢзҡ„ index.html еҸҜд»ҘзңӢеҲ° body еқ—еҶ…е®ҡд№үдәҶпјҡ

<div id="root"></div>

index.html еҶ…еҸҜд»ҘдҪҝз”Ё %PUBLIC_URL% иЎЁзӨә public зӣ®еҪ•ең°еқҖгҖӮ

еҲ йҷӨ src зӣ®еҪ•еҶ…жүҖжңүж–Үд»¶пјҢе№¶ж–°е»ә index.js ж–Үд»¶гҖӮ

иҝҗиЎҢзЁӢеәҸпјҡ

npm start

дјҡиҮӘеҠЁеңЁжөҸи§ҲеҷЁжү“ејҖйЎ№зӣ®гҖӮ

hello world

жңҖз®ҖеҚ•зҡ„ React зӨәдҫӢеҰӮдёӢпјҢе°Ҷд»Јз ҒеҠ е…Ҙ index.js ж–Үд»¶пјҡ

const React = require('react')
const ReactDOM = require('react-dom')

ReactDOM.render(
    <h1>hello world</h1>,
    document.getElementById('root')
);

дҝқеӯҳж–Үд»¶еҗҺеҲ·ж–°жөҸи§ҲеҷЁйЎөйқўжҹҘзңӢж•ҲжһңгҖӮ
