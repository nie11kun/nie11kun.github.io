消除提取出物体的白边 in Photoshop

  • 作者: admin
  • 时间: 2019-12-07
  • 分类: photoshop

使用选区工具选出的物体，很大可能性会有一周的多余的边沿部分，可以使用一下方法消除。

使用你熟悉的选区工具，将主体选出来，可以使用 2020 的 object select tool 进行选取：

建立图层蒙版，可以发现选区周围的多余部分：

选中图层蒙版，点击 filter - other 有两个选项 maximum 和 minimum：

maximum 可以将蒙版选区扩大，minimum 可以将蒙版选区缩小：

我们这里需要用到 minimum，设置里选择 roundness 使过渡圆滑，调节滑块得到最好结果：
