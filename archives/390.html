Photoshop image assets 快生成图片

2019-10-07 photoshop 0
这篇文章上次修改于 818 天前，可能其部分内容已经发生变化，如有疑问可询问作者。

打开 psd 项目文件，选择 file - generator - image assets：

选择想要输出的图层或组改名为 .jpg 或 .png 或 .gif 后缀，则会自动在项目文件目录下生成文件夹，包含有 jpg 或 png 文件：


可以使用逗号 “,” 来生成多个文件，用百分号 “%” 修改生成文件大小：

每次对图层进行修改都会自动重新生成新的图片，这样可以方便的生成多个不同需求的样式。
