使用 Layer Comps  实现一个图片文件多个效果版本 in Photoshop

  • 2020-02-12
  • photoshop
  • 暂无评论

一张图片 psd 项目可以随时切换多种风格的效果，可以在 psd 源文件内将多个风格的图层都建立起来，在 layer comps 做标记，然后在其他项目中调用此 psd 源文件时候就可以选择性的切换各种效果。

想在项目中插入另一个 psd 文件，选择 file - place linked：

插入后，打开右侧 properties 选项，如果没有，从 Windows 里找到打开：

如果打开的 psd 文件里已经定义了 payer comps，则从这个选项就可以切换了：

下面介绍如何定义 layer comps

首先将图片需要的各种风格的图层都建立好：

选择 window - layer comps：

根据想要的第一种风格，关闭打开相关图层，点击 layer comps 下面的加号，将所有选项都打勾，起一个名字，就完成了第一个 layer comps 的定义，点击确定，我这里第一个风格是原始图片：

同理，建立想要的不同风格 layer comps：

在前面的项目中打开我们刚才建立的包含 layer comps 的 psd 文件，选择 file - place linked：

打开 properties 查看可选的 layer comps：

点击不同的风格，可以看到效果了。

以上就是使用 layer comps 实现 psd 文件多种风格的定义。
