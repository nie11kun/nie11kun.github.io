Linux 软链接 link

  • 作者: admin
  • 时间: 2019-10-16
  • 分类: Linux
  • 评论

类似于 Windows 下的快捷方式，Linux 下可以创建一个文件/文件夹的 link，用于某个文件/文件夹的快速访问，也可以用在自己安装的 app 能够使用命令调用，或者设置同一 app 不同版本的调用。

ln -s source_filename symbolic_filename

-s：软连接

Linux 设置开机自启动 python/bash 脚本

  • 作者: admin
  • 时间: 2019-10-15
  • 分类: Linux
  • 评论

打开 rc.local 文件：

vim  /etc/init.d/rc.local

在 rc.local 后加入启动命令：

sudo python /your/path/script.py &

&符号为让脚本后台运行，不在命令行显示结果.
