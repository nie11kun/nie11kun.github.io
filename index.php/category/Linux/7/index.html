修改SSH端口

  • 作者: admin
  • 时间: 2019-03-21
  • 分类: Linux
  • 评论

ssh 默认端口为 22， 有时候为了安全，需要修改默认端口到任意其他端口。

使用 root 用户登录服务器。

ssh [email protected]

打开配置文件：

vim /etc/ssh/sshd_config

找到并修改以下行：

Port 22

修改 22 为任意可用端口，保存并退出。

重启服务：

service ssh restart

重新登录 ssh：

ssh [email protected] -p Port

以上就是修改 ssh 端口的过程。
