Linux kernel 内核相关

  • 作者: admin
  • 时间: 2019-11-17
  • 分类: Linux
  • 评论

Linux内核（英语：Linux kernel），是一种开源的类Unix操作系统宏内核。

如果你在安装新硬件后遇到：网卡不能使用，亮度不能调节，触摸板不能识别，蓝牙不能使用等等，这些关系都和linux的内核有关系。Kernel 是 Linux 系统的核心，主要负责硬件的支持。Linux 内核提供了安全补丁，bugfix 和新特性。
