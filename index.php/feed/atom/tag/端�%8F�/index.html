使用命令如下：

# 显示所有端口情况
netstat -ano

# 显示某个端口的占用情况 返回的数据中最后的数字为 PID 号
netstat -aon|findstr "7200"

# 根据 PID 号结束对应进程
taskkill /f /pid  7200
