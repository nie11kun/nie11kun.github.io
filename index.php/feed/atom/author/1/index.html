此 Bootloader 暂取名为 Breed，不是 U-Boot，也不是 U-Boot 的改进版，是全新、独立的、跟 U-Boot 平级的 Bootloader。

科普一下：
Bootloader 意思为引导加载器，即为用于加载操作系统的程序。它是一大类此类功能程序的统称。现在的 BIOS、UEFI、GRUB、RedBoot、U-Boot、CFE、Breed 等都是 Bootloader。

Breed 不支持启动非 Linux 类型的固件，例如 TP/水星/迅捷的 VxWorks 系统。因此如果固件大小小于等于 2MB，那就肯定不能刷了。

]]> <![CDATA[Windows 查询端口被哪个程序占用]]> https://niekun.net/index.php/archives/922.html 2020-02-13T16:53:00+08:00 2020-02-13T16:53:00+08:00 admin https://niekun.net 调试远程控制的时候，发现 3389 端口被占用了，需要查询到哪个程序占用了此端口。

Linux 查看端口占用参考：https://niekun.net/index.php/archives/430.html

使用命令如下：

# 显示所有端口情况
netstat -ano

# 显示某个端口的占用情况 返回的数据中最后的数字为 PID 号
netstat -aon|findstr "7200"

# 根据 PID 号结束对应进程
taskkill /f /pid  7200
