Marco Nie - 端口 https://niekun.net/index.php/tag/端口/ zh-CN Thu, 13 Feb 2020 16:53:57 +0800 Thu, 13 Feb 2020 16:53:57 +0800 Windows 查询端口被哪个程序占用 https://niekun.net/index.php/archives/922.html https://niekun.net/index.php/archives/922.html Thu, 13 Feb 2020 16:53:57 +0800 admin 调试远程控制的时候，发现 3389 端口被占用了，需要查询到哪个程序占用了此端口。

使用命令如下：

# 显示所有端口情况
netstat -ano

# 显示某个端口的占用情况 返回的数据中最后的数字为 PID 号
netstat -aon|findstr "7200"

# 根据 PID 号结束对应进程
taskkill /f /pid  7200
