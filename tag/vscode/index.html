WSL 下使用 VS Code 编辑器

  • 2020-02-26
  • Windows
  • 暂无评论

Windows Subsystem for Linux (WSL) 就是 Windows 10 下安装的 Linux 子系统，有了它可以很方便的使用 Linux，WSL 也可以很方便的访问本地磁盘文件。

vs code 已经成为目前个平台最流行的代码编辑平台，也是我最常用的编辑器，通过插件设置，可以在 WSL 环境下使用 vs code，这对于环境开发和编辑工作有极大的帮助，下面介绍设置方法。
