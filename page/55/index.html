chown 和 chmod 管理文件权限

  • 作者: admin
  • 时间: 2019-04-19
  • 分类: Linux
  • 评论

Linux 的文件属性主要包含两部分：ownership 和 permission

一个文件或文件夹的属性包括：1.归属于某个 user 和 user group，2.在此 user 和 user group 下的读/写/执行权限及其他 user 的相关权限。分别通过 chown 和 chmod 来进行设置。

参考连接：
https://www.computerhope.com/unix/uchmod.htm
https://www.computerhope.com/unix/uchown.htm

Linux 查看系统信息

  • 作者: admin
  • 时间: 2019-04-19
  • 分类: Linux
  • 评论

free

显示当前内存使用情况，包括物理内存和 swap 虚拟内存

uptime

显示系统当前连续开机时间

top

实时显示系统运行信息，包括内存使用情况及进程信息。

df -h

硬盘可用容量信息

du

查看文件/文件夹空间占用

# 查看当前目录占用空间
du -sh

#查看特定目录占用空间
du -sh /home

pwd

当前目录信息

uname

系统版本信息 内核信息
